## Installation

1. Install node using [nvm](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install
   ```

2. Install dependencies

   ```sh
   npm i
   ```

3. Start the development server

   ```sh
   npm run dev
   ```

## Building

Generate production build

```sh
npm run build
```
